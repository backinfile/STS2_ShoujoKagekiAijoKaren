# 杀戮尖塔2 Mod制作教程

《杀戮尖塔2》原生支持模组，可以直接制作模组并使用。

由于塔2刚发售暂时没有创意工坊，所以目前只能通过本地加载模组。

**注意**：游戏目前处于抢先体验阶段，教程可能不是最优解，且代码更改频繁。
