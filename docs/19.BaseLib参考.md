---
name: SavedSpireField 作用范围
description: BaseLib SavedSpireField 的自动存读档机制只对 AbstractModel 有效
type: reference
originSessionId: a796aaa3-b765-4a9d-bdd8-cbe75427a764
---
`SavedSpireField<TKey, TVal>` 的泛型约束只有 `where TKey : class`，但自动存读档依赖 Patch `SavedProperties.FromInternal` / `FillInternal`。

STS2 游戏本体的 `SavedProperties` 公开入口仅接受 `AbstractModel`：
- `SavedProperties.From(AbstractModel model)`
- `SavedProperties.Fill(AbstractModel model)`

因此虽然理论上手动调用 Internal 方法可对任意对象生效，**在游戏正常运行中 SavedSpireField 基本只对 AbstractModel 及其子类（CardModel/PowerModel/RelicModel 等）有效**。
