## 按机制粗分

如果按机制而不是目录来找，当前卡牌大致可以分成：

- 基础卡
- Shine（闪耀）卡
- PromisePile（约定牌堆）卡
- GlobalMove（全局移牌）联动卡
- Relic / DisableRelic（禁用遗物）卡
- Token / Option 卡
- Curse / Ancient 相关卡

## Power（能力）目录

Karen Power 主要位于：

- `src/Core/Models/Powers/`
- `src/Core/Models/Powers/tmpStrength/`

检索 Power（能力）时，优先从基类、命名和对应机制三个角度找：

1. 是否继承 `KarenBasePower`
2. 名称是否和卡牌 / 机制直接对应
3. 是否在对应系统目录中还有配套扩展代码

## 常见检索方法

当你想找一个现成实现时，优先按以下顺序查：

1. 看 `KarenCardPool.cs`，确认卡是否真的进卡池。
2. 看对应目录名，先定位大类。
3. 搜基类覆写点，例如 `OnPlay`、`CanonicalVars`、`OnUpgrade`。
4. 搜关键系统钩子，例如 `OnAddedToPromisePile`、`OnGlobalMove`。
5. 搜同名 Power、Relic、Patch 是否有联动。
