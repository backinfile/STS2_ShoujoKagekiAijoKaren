# 杂项与运行数据参考

> 主题：不适合单独成篇但经常要查的运行时参考  
> 适用范围：`PlayerChoiceContext`、`RunHistory`、运行期上下文说明  
> 阅读建议：把它当备查页，不作为第一次阅读入口

---

## PlayerChoiceContext

`PlayerChoiceContext` 贯穿：

- 出牌流程
- Hook（钩子）流程
- 选牌与玩家交互
- 联机同步相关选择

写 Mod 时通常不需要自己新建它，而是沿用系统透传下来的 `choiceContext`。

## 适合放在这里的内容

- 运行时上下文说明
- 随机系统、小型参考
- 事件参数说明
- 不足以单独成篇但又常被查阅的技术点

## RunHistory JSON

项目内保留了 `RunHistory` 输出格式参考，用于：

- 对局分析
- JSON 字段对照
- 运行结果上传或本地分析

如果要分析具体字段结构，优先结合当前项目里实际使用 `RunHistory` 的代码一起看。
