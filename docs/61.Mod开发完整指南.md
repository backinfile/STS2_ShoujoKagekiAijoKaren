## 2. 卡牌（CardModel）

### 2.1 基础结构

所有 Karen 卡牌必须继承 `KarenBaseCardModel`：

```csharp
using MegaCrit.Sts2.Core.Commands;
using MegaCrit.Sts2.Core.Entities.Cards;
using MegaCrit.Sts2.Core.GameActions.Multiplayer;
using MegaCrit.Sts2.Core.Localization.DynamicVars;
using MegaCrit.Sts2.Core.Models;
using MegaCrit.Sts2.Core.ValueProps;
using ShoujoKagekiAijoKaren.src.Core.Models.Cards;
using System.Collections.Generic;
using System.Threading.Tasks;

namespace ShoujoKagekiAijoKaren.src.Core.Models.Cards.xxx; // 按类型分目录

public sealed class KarenXxxCard : KarenBaseCardModel
{
    // 构造函数：费用, 类型, 稀有度, 目标类型
    public KarenXxxCard() : base(1, CardType.Attack, CardRarity.Common, TargetType.AnyEnemy) { }

    // 标签（Strike/Defend/PromisePileRelated 等）
    protected override HashSet<CardTag> CanonicalTags => [CardTag.Strike];

    // 动态变量（伤害/格挡/抽牌数等）
    protected override IEnumerable<DynamicVar> CanonicalVars =>
    [
        new DamageVar(6m, ValueProp.Move)
    ];

    // 打出时效果
    protected override async Task OnPlay(PlayerChoiceContext choiceContext, CardPlay cardPlay)
    {
        ArgumentNullException.ThrowIfNull(cardPlay.Target);
        await DamageCmd.Attack(DynamicVars.Damage.BaseValue)
            .FromCard(this)
            .Targeting(cardPlay.Target)
            .WithHitFx(VfxCmd.slashPath)
            .Execute(choiceContext);
    }

    // 升级效果
    protected override void OnUpgrade()
    {
        DynamicVars.Damage.UpgradeValueBy(3m);
    }
}
```

### 2.2 CardType 枚举

| 值 | 说明 |
|---|---|
| `CardType.Attack` | 攻击牌 |
| `CardType.Skill` | 技能牌 |
| `CardType.Power` | 能力牌 |
| `CardType.Curse` | 诅咒牌 |

### 2.3 CardRarity 枚举

| 值 | 说明 |
|---|---|
| `CardRarity.Basic` | 基础牌（初始卡组） |
| `CardRarity.Common` | 普通 |
| `CardRarity.Uncommon` | 罕见 |
| `CardRarity.Rare` | 稀有 |
| `CardRarity.Curse` | 诅咒 |

### 2.4 TargetType 枚举

| 值 | 说明 |
|---|---|
| `TargetType.None` | 无目标 |
| `TargetType.Self` | 自身 |
| `TargetType.AnyEnemy` | 任意敌人 |
| `TargetType.AllEnemies` | 全体敌人 |

### 2.5 CardKeyword 关键词

在 `CanonicalKeywords` 中声明：

```csharp
public override IEnumerable<CardKeyword> CanonicalKeywords => 
    [CardKeyword.Exhaust]; // 消耗
```

| 关键词 | 效果 |
|---|---|
| `CardKeyword.Exhaust` | 打出后消耗（进入消耗堆） |
| `CardKeyword.Ethereal` | 虚无（回合结束弃置） |
| `CardKeyword.Unplayable` | 不可打出 |
| `CardKeyword.Eternal` | 灵魂绑定（无法从牌组移除） |
| `CardKeyword.Innate` | 固有（战斗开始加入手牌） |
| `CardKeyword.Retain` | 保留（回合结束保留在手牌） |

### 2.6 关键属性

```csharp
public override bool GainsBlock => true;          // 打出时获得格挡（必须加，否则格挡不显示悬浮提示）
public override int MaxUpgradeLevel => 0;         // 不可升级（诅咒牌等）
public override int MaxUpgradeLevel => int.MaxValue; // 可无限升级
protected override bool HasEnergyCostX => true;   // X费卡牌
```

### 2.7 KarenBaseCardModel 扳机

```csharp
// 此牌被放入约定牌堆时触发

// 此牌从约定牌堆取出/弃置/清场时触发

// 回合结束时此牌在约定牌堆中触发（Void模式下不触发）

// 闪耀耗尽时触发（仅战斗进行中）
public virtual Task OnShineExhausted(PlayerChoiceContext ctx, bool inCombat, CombatState combatState) => Task.CompletedTask;

// 牌堆间移动时触发
```

```csharp
using MegaCrit.Sts2.Core.Entities.Players;

using MegaCrit.Sts2.Core.GameActions.Multiplayer;

using System.Collections.Generic;
using System.Threading.Tasks;

namespace ShoujoKagekiAijoKaren.src.Models.Relics;

public sealed class KarenXxxRelic : RelicModel
{
    public override RelicRarity Rarity => RelicRarity.Starter; // Starter/Common/Uncommon/Rare

    // 悬浮提示：显示某张卡牌
    protected override IEnumerable<IHoverTip> ExtraHoverTips => 
        [HoverTipFactory.FromCard<KarenTowerOfPromise>(false)];

    // 回合开始抽手牌前
    public override async Task BeforeHandDraw(Player player, PlayerChoiceContext choiceContext, CombatState combatState)
    {
        if (player == base.Owner && combatState.RoundNumber == 2)
        {
            Flash(); // 遗物闪烁
            var card = combatState.CreateCard<KarenTowerOfPromise>(Owner);
            await CardPileCmd.AddGeneratedCardToCombat(card, PileType.Hand, true);
        }
    }
}
```

## 6. 卡池与遗物池

### 6.1 卡池（CardPoolModel）

```csharp
using Godot;
using MegaCrit.Sts2.Core.Models;
using MegaCrit.Sts2.Core.Unlocks;
using System.Collections.Generic;
using System.Linq;

namespace ShoujoKagekiAijoKaren.src.Models.CardPools;

public sealed class KarenCardPool : CardPoolModel
{
    public override string Title => "karen";
    public override string EnergyColorName => "karen";
    public override string CardFrameMaterialPath => "card_frame_purple";
    public override Color DeckEntryCardColor => new("3EB3ED");
    public override Color EnergyOutlineColor => new("1D5673");
    public override bool IsColorless => false;

    protected override CardModel[] GenerateAllCards()
    {
        return
        [
            ModelDb.Card<KarenStrike>(),
            ModelDb.Card<KarenDefend>(),
            // ... 所有卡牌
        ];
    }

    protected override IEnumerable<CardModel> FilterThroughEpochs(UnlockState unlockState, IEnumerable<CardModel> cards)
    {
        return cards.ToList();
    }
}
```

**重要**：凡是加入 `StartingDeck` 的卡，**必须同时加入** `KarenCardPool.GenerateAllCards()`，否则预加载时会崩溃。

### 7.6 动态变量语法

| 语法 | 说明 |
|---|---|
| `{Damage:diff()}` | 显示伤害值及其升级变化 |
| `{Block:diff()}` | 显示格挡值及其升级变化 |
| `{Cards:diff()}` | 显示卡牌数量及其升级变化 |
| `{Energy:energyIcons()}` | 显示能量图标（不加文字） |
| `{StrengthPower:diff()}` | Power 类名必须是完整类名 |
| `{IfUpgraded:show:文本\|}` | 升级后才显示的文本 |
| `{Repeat:diff()}` | 重复次数 |
| `{Heal:diff()}` | 治疗量 |
| `{CalculatedDamage:diff()}` | 计算后的伤害 |
| `{ExtraDamage:diff()}` | 额外伤害 |
| `[gold]关键词[/gold]` | 金色高亮 |
| `[blue]数值[/blue]` | 蓝色高亮 |
| `[red]警告[/red]` | 红色高亮 |
| `[green]增益[/green]` | 绿色高亮 |

**中文格式注意**：变量与中文字符之间**不加空格**；badge（Exhaust/Retain/Innate）**不写入** description，由系统根据 `CanonicalKeywords` 自动添加。

**英文格式注意**：`\n[gold]Shine[/gold]` 需分行；**不用** `[gray]...[/gray]` 表示未升级内容，改用 `{IfUpgraded:show:...|}`。

```csharp
protected override void OnUpgrade()
{
    DynamicVars.Damage.UpgradeValueBy(3m + _upgradeCount);
    _upgradeCount++;
}

// 降级时重置（回退升级）
protected override void AfterDowngraded()
{
    DynamicVars.Damage.ResetToBase();
    for (int i = 0; i < _upgradeCount; i++)
    {
        DynamicVars.Damage.UpgradeValueBy(3m + i);
    }
}
```
