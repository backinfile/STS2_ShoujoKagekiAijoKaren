# 02 风格原画绘制

## 引言

本教程介绍塔2美术资源制作的内容。

- 塔2还处于测试阶段，本教程会不定期更新
- 面向有一定美术基础，想要尝试手绘美术资产的同志
- 0基础也可以尝试，画画本身是件很快乐的事情

## 准备工具

### 绘图工具

1. **数位板/笔** - 最经典的板绘套装，百元级别即可，对腰和颈椎友好
2. **数位屏/笔** - 比数位板好控制，但价格昂贵，便宜的会有色差
3. **iPad/Apple Pencil** - 最适合新手，颜色和操作都很流畅

### 绘图软件

1. **PS/PhotoShop** - 最经典，正版 88元/月，盗版会强制退出
2. **CSP** - 专业板绘软件，正版 429元一次性付费
3. **Krita** - 免费开源，长期更新，稍有 BUG
4. **Procreate** - 仅限 iPad，88元，简单易上手

> 软件之间切换很容易，学会一个后其他的只需熟悉几天即可。

## 塔2美术风格绘制

### 第一步：收集参考

1. 在游戏中截屏作为参考，保证画风统一
2. 找到想要绘制角色的美术资源
3. 推荐使用 `PureRef2.0` 参考图软件

### 第二步：勾勒草稿

1. 将游戏截屏透明度调低
2. 在上面勾勒草稿
3. 确定角色与敌人之间的比例

### 第三步：绘制特征

在草稿基础上画出人物特征：

- **强化服装** - 夸张化关键元素，删去繁琐装饰
- **发型很重要** - 二次元角色靠发型识别，五官次要
- **少用线条** - 塔2偏美式卡通，以色块为主

### 第四步：色块分割

直接大胆地将色块分割好，透视和比例可以有些变形，更符合塔2的漫画风格。

### 第五步：添加细节

用色块切分阴影和亮部，将体积感区分开后稍微增添细节。

## 注意事项

- **衣物不要太多** - 后续需要拆解绑定制作动画，过多毛发和过长衣物会消耗大量精力调试
- **推荐紧身衣或机甲风格** - 方便后续制作
- **不要过度光影** - 塔2游戏内很少有强烈光影，明暗对比强烈反而违和
