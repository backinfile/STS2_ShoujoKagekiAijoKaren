# 05 添加新人物

添加新人物是较为复杂的任务，需要创建多个组件。

## 依赖 BaseLib

参考上一章添加对 `BaseLib` 的依赖，这可以省去很多功夫。

## 创建池子

需要创建人物独有的卡牌、药水、遗物池各一个。

### 卡牌池

`TestCardPool.cs`:

```csharp
public class TestCardPool : CustomCardPoolModel
{
    // 卡池的ID，必须唯一防撞车
    public override string Title => "test";

    // 卡池的能量图标，暂时不支持加载，建议使用原版
    public override string EnergyColor => "red";
}
```

### 药水池

```csharp
public class TestPotionPool : CustomPotionPoolModel
{
    public override string Title => "test";
}
```

### 遗物池

```csharp
public class TestRelicPool : CustomRelicPoolModel
{
    public override string Title => "test";
}
```

## 创建人物类

```csharp
[Pool(typeof(CharacterPool))]
public class TestCharacter : CustomCharacterModel
{
    public override string Id => "TEST_CHARACTER";

    public override int StartingHp => 75;

    public override Type CardPoolType => typeof(TestCardPool);
    public override Type RelicPoolType => typeof(TestRelicPool);
    public override Type PotionPoolType => typeof(TestPotionPool);

    // 其他配置...
}
```

## 添加起始卡牌和遗物

在人物类中配置：

```csharp
public override List<string> StartingDeck => new()
{
    "STRIKE",
    "STRIKE",
    "STRIKE",
    "DEFEND",
    "DEFEND"
};

public override List<string> StartingRelics => new()
{
    "STARTER_RELIC"
};
```

## 自定义人物模型

### 使用 Spine 动画

参考视觉教程中的 Spine 导入部分。

### 自定义背景

创建继承 `NCreatureVisuals` 的场景，包含以下节点：

- `Visuals(Node2D)` - 唯一化命名(%)
- `Bounds(Control)` - 唯一化命名(%)
- `IntentPos(Marker2D)` - 唯一化命名(%)
- `CenterPos(Marker2D)` - 唯一化命名(%)

## 注意事项

- 人物 ID 必须唯一
- 确保所有池子都正确注册
- 测试起始卡组和遗物是否正常
