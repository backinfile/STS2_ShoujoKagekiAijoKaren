# 03 添加新内容

本章介绍如何添加新的游戏内容。

## 项目结构

创建以下基本结构：

```
YourMod/
├── Scripts/
│   ├── Entry.cs          # 模组入口
│   ├── Cards/            # 卡牌
│   ├── Relics/           # 遗物
│   └── ...
├── Resources/
│   ├── images/           # 图片资源
│   └── localization/     # 本地化文本
├── yourmod.json          # 模组配置
└── yourmod.csproj        # 项目配置
```

## 添加新卡牌

### 1. 创建卡牌类

```csharp
using MegaCrit.Sts2.Core.Cards;
using MegaCrit.Sts2.Core.Pools;

namespace YourMod.Cards;

public class MyNewCard : CardModel
{
    public override string Id => "MY_NEW_CARD";

    // 其他属性配置...
}
```

### 2. 注册到卡池

使用 `ModHelper.AddModelToPool` 注册卡牌：

```csharp
ModHelper.AddModelToPool<ColorlessCardPool>(new MyNewCard());
```

### 3. 添加本地化文本

在 `localization/zh-CN.json` 中添加：

```json
{
  "MY_NEW_CARD.title": "我的新卡牌",
  "MY_NEW_CARD.description": "造成{Damage:diff()}点伤害。"
}
```

## 添加卡图

### 方法1：直接指定路径

在卡牌类中 override `PortraitPath`：

```csharp
public override string PortraitPath => "res://yourmod/images/my_card.png";
```

### 方法2：使用 Harmony Patch

```csharp
[HarmonyPatch(typeof(CardModel), nameof(CardModel.PortraitPath), MethodType.Getter)]
public static class CardModel_GetPortrait_Patch
{
    private static readonly Dictionary<string, string> CustomPortraits = new()
    {
        ["MyNewCard"] = "res://yourmod/images/my_card.png",
    };

    static void Postfix(CardModel __instance, ref string __result)
    {
        var className = __instance?.GetType().Name;
        if (string.IsNullOrEmpty(className)) return;
        if (!CustomPortraits.TryGetValue(className, out var path)) return;
        if (!ResourceLoader.Exists(path)) return;
        __result = path;
    }
}
```

## 添加新遗物

类似于添加卡牌，创建继承 `RelicModel` 的类并注册到遗物池。

## 打包模组

编译后将以下文件放入 `mods` 文件夹：

- `yourmod.dll` - 编译生成的程序集
- `yourmod.pck` - Godot 打包的资源文件
- `yourmod.json` - 模组配置文件
