# 01 环境配置

## 安装 .NET SDK

前往 https://dotnet.microsoft.com/zh-cn/download 下载并安装 .NET 8.0 SDK。

## 安装 Godot

前往 https://godotengine.org/download/ 下载 Godot 4.3.stable.mono 版本。

**注意**：必须是 mono 版本，否则无法使用 C# 编程。

## 创建项目

1. 打开 Godot，创建一个新项目
2. 选择 C# 作为编程语言
3. 项目名称建议使用英文

## 配置 IDE

推荐使用以下 IDE 之一：

- Visual Studio 2022
- JetBrains Rider
- Visual Studio Code

配置方法参考 Godot 官方文档。
