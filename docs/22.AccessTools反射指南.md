# AccessTools 反射指南

> 基于 HarmonyLib 的 `AccessTools` 和 `Traverse`，用于在 Mod 中访问游戏本体的私有字段/方法。

---

## 1. 获取私有字段值

```csharp
using HarmonyLib;

// 获取实例字段
var value = AccessTools.Field(typeof(目标类), "_字段名").GetValue(实例对象);

// 获取并转换类型
var label = AccessTools.Field(typeof(NPower), "_amountLabel").GetValue(__instance) as MegaLabel;
```

**项目中实际用法**：
```csharp
// src/Core/Patches/NPowerDisplayPatch.cs
var amountLabel = AccessTools.Field(typeof(NPower), "_amountLabel").GetValue(__instance) as MegaLabel;
```

---

## 2. 设置私有字段值

```csharp
// 直接设置
AccessTools.Field(typeof(目标类), "_字段名").SetValue(实例对象, 新值);

// 或用 Traverse（更推荐）
Traverse.Create(实例对象).Field("_字段名").SetValue(新值);

// 泛型版本（类型安全）
Traverse.Create(实例对象).Field<字段类型>("_字段名").Value = 新值;
```

---

## 3. 调用私有方法

```csharp
// 获取方法信息
var method = AccessTools.Method(typeof(目标类), "方法名", new[] { typeof(参数1类型), typeof(参数2类型) });

// 调用实例方法
var result = method.Invoke(实例对象, new object[] { 参数1, 参数2 });

// 调用静态方法
var result = method.Invoke(null, new object[] { 参数1, 参数2 });
```

**项目中实际用法**：
```csharp
// src/Core/Patches/NPowerRefreshPatch.cs
var setTextMethod = AccessTools.Method(amountLabel.GetType(), "SetTextAutoSize", new[] { typeof(string) });
setTextMethod?.Invoke(amountLabel, new object[] { newText });
```

---

## 4. Traverse 链式操作

```csharp
// 读取嵌套私有字段
var value = Traverse.Create(对象A)
    .Field("_对象B")
    .Field("_对象C")
    .Field<int>("_数值字段")
    .Value;

// 设置嵌套私有字段
Traverse.Create(对象A)
    .Field("_对象B")
    .Field("_对象C")
    .Field<int>("_数值字段")
    .Value = 100;
```

**项目中实际用法**：
```csharp
// src/Core/Patches/GameOverScreenCrashPatch.cs
RunState runState = Traverse.Create(__instance).Field<RunState>("_runState").Value;
```

## 5. 结合 Godot 使用注意事项

### 问题：Godot `.tscn` 无法序列化跨程序集的 `private [Export]` 字段

`NParticlesContainer._particles` 是 `private [Export]`，定义在 `sts2.dll` 中。`.tscn` 加载时 Godot 尝试赋值会报错：
```
Failed to get property '_particles' from node 'xxx'
```

### 解决方案：`.tscn` 去掉 `node_paths`，`_Ready()` 里反射赋值

```tscn
# 错误写法（Godot 无法访问跨程序集 private 字段）
[node name="xxx" type="Node2D" node_paths=PackedStringArray("_particles")]
_particles = [NodePath("GPUParticles2D")]

# 正确写法
[node name="xxx" type="Node2D"]
# 不写 node_paths，不写 _particles

[node name="GPUParticles2D" type="GPUParticles2D" parent="."]
```

```csharp
public override void _Ready()
{
    // 手动收集子节点
    var particles = new Godot.Collections.Array<GpuParticles2D>();
    foreach (var child in GetChildren())
        if (child is GpuParticles2D p)
            particles.Add(p);

    // 反射给父类 private 字段赋值
    Traverse.Create(this).Field<Godot.Collections.Array<GpuParticles2D>>("_particles").Value = particles;
}
```

## 6. 完整示例：SNParticlesContainer

```csharp
using Godot;
using HarmonyLib;
using MegaCrit.Sts2.Core.Nodes.Vfx.Utilities;

namespace ShoujoKagekiAijoKaren.Core;

[GlobalClass]
public partial class SNParticlesContainer : NParticlesContainer
{
    public override void _Ready()
    {
        var particles = new Godot.Collections.Array<GpuParticles2D>();
        foreach (var child in GetChildren())
        {
            if (child is GpuParticles2D p)
                particles.Add(p);
        }
        Traverse.Create(this).Field<Godot.Collections.Array<GpuParticles2D>>("_particles").Value = particles;
    }
}
```

## 7. 关键要点

| 场景 | 推荐方式 |
|------|---------|
| Patch 中访问私有字段 | `AccessTools.Field` 或 `Traverse.Create` |
| 需要链式访问多层私有字段 | `Traverse` |
| 需要调用私有方法 | `AccessTools.Method` + `Invoke` |
| `.tscn` 无法序列化的 private [Export] | `.tscn` 去掉赋值 + `_Ready()` 反射 |
| 静态字段/方法 | `AccessTools.Field/Method` + `null` 作为实例参数 |
