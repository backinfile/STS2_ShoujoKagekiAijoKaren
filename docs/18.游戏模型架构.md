## 一、核心基类 AbstractModel

```csharp
public abstract class AbstractModel : IComparable<AbstractModel>
{
    public ModelId Id { get; }                          // 唯一标识
    public bool IsMutable { get; private set; }        // 是否为可变实例
    public bool IsCanonical => !IsMutable;              // 是否为原版实例

    // 核心克隆机制
    public AbstractModel ClonePreservingMutability()    // 保持可变性的克隆
    protected AbstractModel MutableClone()              // 创建可变副本
    protected virtual void DeepCloneFields()            // 深拷贝字段（子类重写）

    // 100+ 个战斗/游戏事件钩子（虚方法）
    public virtual Task BeforeCombatStart() { }
    public virtual Task AfterCardPlayed(PlayerChoiceContext ctx, CardPlay play) { }
    public virtual decimal ModifyDamageAdditive(...) { return 0m; }
    // ... 更多钩子
}
```

### 设计模式

| 模式 | 说明 |
|------|------|
| **Canonical/Mutable** | 原版模型（IsCanonical=true）全局唯一；可变副本用于游戏运行时 |
| **Hooks 系统** | 大量虚方法允许子类介入游戏流程的各个阶段 |
| **Source Generation** | 使用 `[GenerateSubtypes]` 自动生成子类型注册 |

// 保存
public SerializableXxx ToSerializable()
{
    AssertMutable();
    return new SerializableXxx
    {
        Id = base.Id,
        Props = SavedProperties.From(this),  // 反射保存标记[SavedProperty]的字段
        // ... 其他字段
    };
}

// 加载
public static XxxModel FromSerializable(SerializableXxx save)
{
    XxxModel model = SaveUtil.XxxOrDeprecated(save.Id).ToMutable();
    save.Props?.Fill(model);  // 反射填充
    return model;
}

## 文档内容概要

### 1. 卡牌 (CardModel)
- 构造函数: `base(energyCost, CardType, CardRarity, TargetType)`
- 攻击命令链: `DamageCmd.Attack().FromCard().Targeting().Execute()`
- 关键词: `CardKeyword.Exhaust/Ethereal/Innate/Retain`
- 升级: `base.DynamicVars.Damage.UpgradeValueBy(2m)`

### 2. 能力 (PowerModel)
- **StackType**: `Counter`(叠加) / `Single`(单一)
- **Type**: `Buff` / `Debuff`
- 伤害修改: `ModifyDamageAdditive()` / `ModifyDamageMultiplicative()`
- 格挡修改: `ModifyBlockMultiplicative()`
- 回合触发: `AfterTurnEnd()` / `AfterSideTurnStart()`

### 3. 遗物 (RelicModel)
- 常用钩子: `AfterCombatVictory`, `OnEnterRoom`, `OnDrawCard`, `ModifyIncomingDamage`
- 闪烁: `Flash()`

### 4. Command API 汇总
| 命令 | 用途 |
|------|------|
| `DamageCmd.Attack()` | 攻击伤害 |
| `CreatureCmd.GainBlock()` | 获得格挡 |
| `PowerCmd.Apply<T>()` | 应用能力 |
| `CardPileCmd.Draw()` | 抽牌 |
| `CreatureCmd.Heal()` | 治疗 |
| `CardSelectCmd.FromHand/Discard/Draw()` | 选牌界面 |

### 5. DynamicVar 类型
- `DamageVar/BlockVar` - 伤害/格挡值
- `PowerVar<T>` - 能力层数
- `EnergyVar/CardsVar` - 能量/卡牌数量
- `HpLossVar/HealVar` - 失去生命/治疗
