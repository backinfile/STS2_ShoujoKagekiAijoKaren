## 附录：关键词说明

| 关键词 | 说明 |
|--------|------|
| **闪耀（Shine）** | 卡牌每次打出 Shine-1，归零时触发效果并从卡组移除 |
| **约定牌堆（Promise Pile）** | 特殊的牌堆，与手牌、抽牌堆、弃牌堆独立 |
| **沉淀** | 被打出之前耗能变为0 |
| **保留（Retain）** | 回合结束时保留在手牌中 |
| **虚无（Ethereal）** | 回合结束时如果没有打出则消耗 |
| **固有（Innate）** | 战斗开始时加入手牌 |
| **消耗（Exhaust）** | 打出后进入消耗堆，不再进入弃牌堆 |
| **不能被打出** | 无法从手牌中打出 |
